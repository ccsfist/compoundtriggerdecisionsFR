# Finalisation des Déclencheurs d’Actions Anticipatoires : Décisions Clés

Le nombre d’années pouvant déclencher est limité par les budgets
	- De nombreux bailleurs de fond ne peuvent financer que **3 années sur 10**

Les premières étapes de conception produisent de nombreux déclencheurs possibles

- Chaque mois peut déclencher dans n’importe quelle année
- Par exemple : 
	- si avril peut déclencher **30 % du temps**
	- et si mai peut aussi déclencher **30 % du temps**
	- **plus de 30 % des années pourraient déclencher**
- Si les prévisions étaient complètement aléatoires :
	- pour 6 mois, 
	- chacun réglé à 30 %,
	- on s’attendrait à ce que **plus de 80 % des années déclenchent**
	- Nous nous attendons à moins, car les prévisions ne sont pas totalement aléatoires

- **Les différentes stratégies pour réduire la fréquence ont des niveaux de précision différents** 
	- Certaines stratégies peuvent **augmenter la précision** 
	- Certaines stratégies peuvent **déclencher plus tôt** dans l’année, offrant un meilleur délai d’anticipation

Ainsi, les choix de conception finaux, et probablement les plus importants, consistent à choisir une règle permettant de combiner les différents éléments afin de : 

- Réduire la fréquence pour respecter les limites de financement 
- Augmenter la précision du protocole de prévision 

**Dans cette session, nous vous présenterons les options les plus pertinentes et nous vous demanderons quelle solution finalisée vous semble la meilleure.**
